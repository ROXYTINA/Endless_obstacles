# itch.io Setup Checklist - Endless Obstacle

Quick checklist to publish your game on itch.io for portfolio display.

---

## ✅ Pre-Setup (Do This First)

- [ ] Have your game executable ready: `EndlessObstacle-windows.zip`
- [ ] Create screenshots or GIFs of gameplay (3-5 images)
- [ ] Write game description
- [ ] Create itch.io account at https://itch.io/register

---

## 📋 Step-by-Step Setup

### 1️⃣ Create Game Project

```
Homepage → Create → New Project
```

| Field | Value |
|-------|-------|
| **Project Title** | `Endless Obstacle` |
| **Project URL** | `endless-obstacle` |
| **Classification** | Game |
| **Kind of project** | Windows Executable |

**Click: SAVE**

---

### 2️⃣ Fill Game Details

```
Edit Project → Edit Game
```

**Description Section:**
```
An endless runner game featuring Naruto characters! 

Survive increasingly difficult obstacles, collect points, 
and beat your high score. Features smooth 60 FPS gameplay 
with multiple characters to choose from.

🎮 Features:
• 60 FPS smooth gameplay
• 3 playable characters (Naruto, Sasuke, Sakura)
• Progressive difficulty
• Score persistence
• Dynamic audio effects

🕹️ Controls:
← → Arrow Keys to move
Space to jump
ESC to pause
Mouse to navigate menus
```

**Tags:**
```
naruto, endless-runner, arcade, action, 2d-platformer, java
```

**Credit & Links:**
- Portfolio: https://github.com/ROXYTINA
- GitHub: https://github.com/ROXYTINA/Endless_obstacles
- Author: ROXYTINA

**Click: SAVE GAME**

---

### 3️⃣ Upload Game Files

```
Edit Project → Uploads
```

**Click: UPLOAD FILES**

1. Select file: `EndlessObstacle-windows.zip`
   - Display as: (leave default)
   - Preorder: (uncheck)
   - Click: **UPLOAD**

2. After upload completes:
   - Check: ✅ "This file will be downloaded"
   - Mark as: Select **"Windows executable"** from dropdown
   - Click: **SAVE**

---

### 4️⃣ Add Screenshots

```
Edit Project → Uploads
```

**Add Cover Image:**
- This is the thumbnail shown on your game page
- Click: ADD COVER IMAGE
- Choose a good screenshot or logo

**Add Thumbnail:**
- 500×500 px recommended
- Shows on game listings

**Add Screenshots:**
- Add 3-5 gameplay screenshots
- JPG or PNG format
- Different scenes/states (menu, playing, game-over)

---

### 5️⃣ Configure Release Settings

```
Edit Project → Danger Zone / Visibility
```

| Setting | Value |
|---------|-------|
| **Visibility** | Public |
| **Downloadable** | ✅ Yes |
| **Can be purchased** | ❌ No (if not charging) |
| **Mature content** | ❌ No |

**Click: SAVE**

---

### 6️⃣ Publish

```
Edit Project → Basic
```

- ✅ **Published**: Check the box
- Click: **SAVE GAME**

Your game is now LIVE! 🎉

---

## 📤 To Get Embed Code

Use this in your portfolio HTML to embed the game card!

## 🐛 Troubleshooting

**Q: "Players can't download the file"**
- A: Check file upload is marked "This file will be downloaded"

**Q: "Game won't run on their computer"**
- A: Ensure they extract the ZIP before running the .exe
- The .exe has Java runtime bundled, no installation needed

**Q: "Where's my embed code?"**
- A: Click SHARE on your game page (top right)
- Only works for published games

**Q: "Can I update the game?"**
- A: Yes! Go to Uploads → Upload new version
- Players will download latest automatically

**Q: "How do I see download stats?"**
- A: Dashboard → Your Game → Analytics tab
