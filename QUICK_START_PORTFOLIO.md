## 📋 Step-by-Step: The Best Option

### Phase 1: Publish to itch.io (15 minutes)

```
1. Go to https://itch.io/register
2. Create account (or sign in if you have one)
3. Click "Create" → "New Project"
4. Fill in:
   - Title: Endless Obstacle
   - URL: endless-obstacle
   - Kind: Windows executable
5. Upload: EndlessObstacle-windows.zip
   - From: D:\My project\Endless Obstacle\platform2D\dist\
6. Add description: (copy from ITCH_IO_SETUP.md)
7. Set status to "Published"
8. Your game is LIVE! 🎉
```

**Result:** Players can download & play from itch.io

### Phase 2: Enable GitHub Pages (10 minutes)

```
1. Go to GitHub repo: https://github.com/ROXYTINA/Endless_obstacles
2. Click Settings (gear icon)
3. Click Pages (left sidebar)
4. Source: main branch / root
5. Click Save
6. Wait 1-2 minutes
7. Your portfolio is LIVE! 🌍
```

**Result:** Portfolio lives at `https://roxytina.github.io/endless-obstacles/`
